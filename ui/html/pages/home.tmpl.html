JetAPI is a lightweight and customizable API that provides aircraft information in JSON format by scraping publicly available data. Currently, it sources images and flight details from JetPhotos and FlightRadar24.

Refer to the documentation to learn how to structure a query. You can also use the Query Builder to build a query and preview its JSON structure.

The general search in the top right shows how data from the API is used to generate a simple page with details about a specific aircraft.
